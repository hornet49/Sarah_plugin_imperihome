SARAH communique avec la box domotique Vera lite

Version expérimentale !

N'oubliez pas de mettre l'IP de votre vera dans le fichier veralite.prop!

Ce plugin permet les actions suivantes:

  • Allumer ou éteindre un module ON/OFF sur la Véra
  • commander un module Virtual Switch sur la Véra
  • Modifier la luminosité d'un module variateur
  • Controler un volet
  • Déclencher une scène par son numéro
  • Interroger un capteur
  • Interroger l'état d'un module ON/OFF

Example de commande a donner :

  • Sarah allume la lampe du salon s'il te plait
  • Sarah quel est la température de la piscine ?
  • Sarah ouvre le portail
  • Sarah règle la lumière du couloir à 60%

La grammaire et les modules de la Véra sont déclarés dans le fichier veralite.xml

En fonction des phrases, on distingue les actions "SET" des interrogations "GET"

Chaque module de la vera est codé suivant son periphId et un numéro de périphérique:

  • periphType="1" : Module ON/OFF
  • periphType="2" : Variateur de lumière
  • periphType="3" : Scene
  • periphType="4" : Sonde de température
  • periphType="5" : Sonde d'hygromètrie
  • periphType="6" : Sonde de luminosité
  • periphType="7" : Sonde de Consommation
  • periphType="8" : Module DoorLock
  • periphType="9" : Volets
  • periphType="10" : Virtual Switch
